# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Overview

This is a text-only repository containing prayers and devotional writings related to Charles Spurgeon. There is no code, build system, or test suite.

## Structure

- `txt/` — All content lives here as plain `.txt` files
  - **Numbered prayers (01–50):** Individual themed prayers/devotionals (e.g., "The Valley Of Vision", "Confession And Petition", "Morning Dedication"). Each file is titled on the first line followed by the text.
  - **Compiled volumes (jcprayer1–5):** Larger compilations that aggregate the individual prayers along with scripture references, commentary, and sermon excerpts.
  - File 43 exists in two variants: `God'sUnspeakableGift.txt` (with apostrophe) and `GodsUnspeakableGift.txt` (without).

## Format Conventions

- Plain text, no markup
- Lines are soft-wrapped at roughly 70 characters
- Titles appear on the first line, followed by a blank line, then body text
